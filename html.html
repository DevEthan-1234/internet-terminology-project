Hypertext Markup Language (HTML)

What is HTML?

HTML is the standard markup language used to create the structure of web pages. It uses "tags" to annotate text, images, and other content so that web browsers know how to display them. While CSS handles the style and JavaScript handles the behavior, HTML provides the essential skeleton of the site.

The Basic Structure

An HTML document is made up of nested elements. Key tags include:

  • <html>: The root element that wraps all content.
  • <head>: Contains meta-information (like the title and CSS links).
  • <body>: Contains the visible content like headings, paragraphs, and images.

The Evolution: HTML5

Since its creation by Tim Berners-Lee in 1991, HTML has evolved significantly. The current version, HTML5, introduced powerful features like native video and audio support, drawing canvases, and better "semantic" tags (like <article> and <footer>) that help search engines understand the content better.
