Uniform Resource Locators (URL)

What is a URL?

A Uniform Resource Locator (URL) is the unique address used to locate a specific resource on the web. It can point to a website, an image, a video, or a document. Think of it as the "digital coordinates" for a specific piece of data on the internet.

Anatomy of a URL

A typical URL is made up of several distinct components that tell the browser how to find and retrieve the resource:

  • Protocol: (e.g., https://) Tells the browser which language to use to communicate.
  • Domain Name: (e.g., www.example.com) The human-readable name of the server hosting the resource.
  • Path: (e.g., /images/logo.png) The specific location of the file on that server.
  • Parameters: (e.g., ?id=123) Extra information passed to the server to find specific data.

URL vs. URI

While the terms are often used interchangeably, a URI (Uniform Resource Identifier) is the broad category, and a URL is a specific type of URI that includes the mechanism (protocol) to find the resource. All URLs are URIs, but not all URIs are URLs!
